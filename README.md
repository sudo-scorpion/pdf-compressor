# Ultimate PDF Suite 🚀

**Ultimate PDF Suite** is a powerful, privacy-focused, browser-based toolkit designed to handle all your PDF needs without ever sending your files to a server.

## ✨ Features

- **📉 PDF Compression**: Significantly reduce the size of presentation PDFs using advanced rasterization.
- **👁️ OCR (Text Recognition)**: Extract searchable text from PDF slides or images locally using `tesseract.js`.
- **🖼️ PDF to Image**: Convert PDF pages into high-quality JPEG/PNG images instantly.
- **🔗 Merge PDFs**: Combine multiple files into a single, optimized document.
- **🌓 Dark Mode**: A premium, responsive UI that looks beautiful in any lighting.
- **🛡️ Privacy First**: Everything runs in your browser. Your data never leaves your computer.

## 🛠️ Technology Stack

- **Framework**: React 19 + Vite
- **Styling**: Vanilla CSS + TailwindCSS (for layout utilities)
- **Icons**: Lucide React
- **PDF Engine**: pdfjs-dist + pdf-lib
- **OCR Engine**: Tesseract.js

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
